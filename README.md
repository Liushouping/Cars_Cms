## 租貸超跑 形象網站＋後台管理（練習用）

Laravel 8 開發，以 Dcatadmin 作為後台管理 如下:

- Fullpage 響應式選單
- Youtube lighbox 影片燈箱
- SendMail 寄信功能
- Loader Animation 加載動畫
- Back To Top 回到首頁
- Lang 切換語言 / 英文 / 中文

後台功能 如下：

- 首頁圖片
- 最新消息

- 超跑車款
- 性能車款

- 租車流程
- 租車須知

- 工作團隊
- 網站配置（ Meta Seo / Facebook Seo ）
